MyCuisine Private Table v4

What this version includes:

1. Luxury Tuscan / Michelin theme
   Light and dark mode are included.

2. Split files like your company site
   index.html = main recipe website
   admin.html = admin portal
   recipes.js = data file
   logic.js = shared safety and scoring logic
   app.js = main site behavior
   admin.js = admin portal behavior
   styles.css = full theme/design
   assets/ = MyCuisine logos and emblems

3. Default access
   Friends access code: kitchen2026
   Admin password: mycuisine2026

4. Admin shortcut
   Mac: Command + Option + A
   Windows: Ctrl + Alt + A

5. T.J.S profile
   T.J.S is set as lactose intolerant.
   Blocked examples include milk, butter, cream, cheese, yogurt, whey, casein, lactose, milk powder, alfredo, ranch, queso, and hidden dairy wording.

6. Added features
   Friend profiles
   Cooking-for group mode
   Recipe approval queue
   Better automatic recipe safety logic
   Safety score system
   Meal planner
   Auto Plan My Week
   Cook This Tonight
   Pantry matching
   Grocery list sorted by store section
   Smart substitutions
   Recipe versions
   Favorite collections
   Photo card support through image URL
   Nutrition fields
   Mobile cooking mode

How to upload to GitHub Pages:

1. Unzip this folder.
2. Upload/replace all files in your GitHub repository.
3. Keep the file names the same.
4. Open index.html through GitHub Pages.
5. Use admin.html to make changes.
6. After admin edits, export recipes.js from the admin portal and replace the existing recipes.js in GitHub.

Important security note:
This is a static GitHub Pages app. The access code and admin password are light gates, not true security. For real friends-only accounts, use Supabase Auth or another real login system later.
